## Personal Blog

### A simple personal blog to write and publish articles on various topics.

## Usage

### Install packages

```bash
python -m pip install -r requirements.txt
```

### Run it locally
```bash
python personal-blog.py
```
> then, visit in the browser the url: http://127.0.0.1:8080/admin

> After creating, editing and deleting your blogs. You can visit http://127.0.0.1:8080/home to see your articles.
